# ButterOS

ButterOS is a barebones "operating system" made with vanilla web technologies. This project is no longer actively maintained and currently has a limited feature set.

## Features

- Window management: Open, close, minimize, maximize, and move windows freely.
- Da Hub service integration: Directly integrates with Da Hub by loading the app list from there into the app drawer.
- Settings app: You can change your background. It's not much, but at least it looks pretty.
